# Decision Tree and Random Forest Practice

## Project Description

This notebook is a practice project for learning Decision Tree and Random Forest classification using Scikit-learn.

A synthetic classification dataset is generated using Scikit-learn's `make_classification()` function and prepared for further machine learning experimentation.

---

## Dataset

Dataset generated using:

```python
make_classification(
    n_samples=1000,
    n_features=4,
    n_classes=2,
    random_state=42
)
```

Dataset Characteristics:

* Number of Samples: 1000
* Number of Features: 4
* Number of Classes: 2
